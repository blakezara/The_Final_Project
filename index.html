Game of Thrones Analysis and Predictions

Final Project for Northwestern Data Science Bootcamp

Using a game of thrones dataset from kaggle I have attempted to create a prediction model for which character will die next. The dataset is a combination of two files I found which is mainy made up of sparse binary data from the books and not the tv show. I explored and cleaned the data using python, pandas, matplotlib and seaborn. In my attempt to find a predictor I used a logisitic regression, a KMeans model and Support Vector Machine model. Though some of my results seemed promising the data is sparse and fictional so it cannot necessarily be expected to produce accurate results. The project was interesting though and might even give spoilers!
